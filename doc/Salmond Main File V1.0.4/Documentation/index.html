If you would like to edit the color, font, or style of any elements in one of these columns, you would do the following:

								
.salmond-navbar ul li a {
    font-family: 'Cabin', sans-serif;
    text-transform: uppercase;
    color: #fff;
    letter-spacing: 0.2em;
    font-weight: 600;
    font-size: 12px;
    padding: 5px 0;
}

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.

I.E.

								
.salmond-one-page-nav .salmond-navbar ul li a {
    font-family: 'Cabin', sans-serif;
    text-transform: uppercase;
    color: #fff;
    letter-spacing: 0.2em;
    font-weight: 600;
    font-size: 12px;
    padding: 5px 0;
}

B) CSS Files and Structure - top

Salmond Template contains two main CSS files. The first one is a generic normalize file. Many browser interpret the default behavior of html elements differently. By using a general normalize CSS file, we can work round this. This file also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.

The second file contains all of the specific stylings for the page. The file is separated into sections using:

								
/*
Generic Area
==========================================================
*/

Generic Area CSS

/*
Header Area
==========================================================
*/

Header Area CSS

/*
Section Area
==========================================================
*/

Section Area CSS

/*
Section Area
==========================================================
*/

Section Area CSS

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

								
.title h1 {
	font-size: 65px;
	text-transform: capitalize;
	line-height: 1.2;
}

D) Video - top

Salmond comes with video format HTML file. In the code below poster is just a image link. This image will show if video loads slowly. You need to change src="video/video.webm" and src="video/video.mp4". In the src destination video is the folder name and video.webm or video.mp4 is video file name. If you want to update the video you need to save your video in two different format. One is .mp4 and the other is .webm. In the src just put the folder & file name with proper extension. 

        						
<video poster="images/slider/3.jpg" id="bgvid" playsinline autoplay muted loop>
    <source src="video/video.webm" type="video/webm">
    <source src="video/video.mp4" type="video/mp4">
</video>

E) JavaScript - top

  • jQuery
  • jQuery 3rd party plugins
  • Salmond custom script
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. Most of the animation in this site is carried out from the customs scripts. There are a few functions worth looking over.
  3. In addition to the custom scripts, We implement a few "tried and true" plugins to create the effects. This plugin is packed, so you won't need to manually edit anything in the file. The only necessary thing to know is how to call the method. For example: 
								
/*
======================================
	theiaStickySidebar Init
======================================
*/
(function() {
    $('.salmond-blog-posts, .widget-area').theiaStickySidebar();
})();


/*
======================================
	ScrollTop Init
======================================
*/
(function() {
	$('.back-to-top').on('click', function(){
		$('html, body').animate({'scrollTop': '0px'}, 2000, "easeInOutExpo");
		return false;
	});
})();
